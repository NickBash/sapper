# Sapper - Сапер

## Пример

Вы можете попробовать игру [здесь](https://sapper-ce94f.web.app)

## Запуск

Вы можете запустить проект

### `npm run start`

И открыть [http://localhost:8080](http://localhost:8080) в браузере

Или собрать

### `npm run build`

Файлы будут в директории /dist
